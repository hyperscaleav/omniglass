Omniglass UI

Design System

Built on daisyUI 5 + Tailwind CSS 4. Two themes, two density modes, two type systems — all driven by data- attributes on <html>. Static components are pure daisyUI markup; interactive components pair daisyUI classes with Kobalte for behavior and accessibility in the SolidJS app.

Components · Interactive

Kobalte + daisyUI

In the SolidJS app, these are Kobalte primitives styled with daisyUI classes. Kobalte owns behavior, focus management, and ARIA; daisyUI owns the look. Each block shows the visual states and maps every Kobalte part to its classes — so the markup drops straight onto Kobalte's anatomy. The open states below are rendered statically for reference.

Focus-visible ring recipe (all parts): focus-visible:outline-none focus-visible:ring-2 focus-visible:ring-primary focus-visible:ring-offset-1 focus-visible:ring-offset-base-100
