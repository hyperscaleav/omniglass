Omniglass release {{ .Release.Name }} is installed in namespace {{ .Release.Namespace }}.

1. Wait for the server to become ready:

     kubectl -n {{ .Release.Namespace }} rollout status deploy/{{ include "omniglass.fullname" . }}

2. Reach the console (until an ingress/exposure is added by the preview pipeline):

     kubectl -n {{ .Release.Namespace }} port-forward svc/{{ include "omniglass.fullname" . }} 8080:{{ .Values.service.port }}

   then open http://localhost:8080/web
