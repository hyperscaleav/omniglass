---
title: Omniglass
description: Open observability and control plane for AV and IT estates, and a place to learn monitoring by operating the real thing.
---

import { LinkCard, CardGrid } from '@astrojs/starlight/components';

Omniglass collects telemetry from devices, types it into owned datapoints, models health
across systems and locations, alarms on it, and acts. It is a single Go binary over a BYO
PostgreSQL database. It is a working monitoring tool that also teaches monitoring: every
operator surface explains the concept it operates on, against real or simulated data, so you
learn the discipline while you run your estate. [View it on GitHub](https://github.com/hyperscaleav/omniglass).

:::note[Early, and public from the first commit]
Omniglass is being rebuilt in the open, one vertical slice at a time. The architecture is
published ahead of the code, so the design is readable before the feature that implements
it lands. Expect the surface to move.
:::
