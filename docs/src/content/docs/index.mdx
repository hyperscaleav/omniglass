Omniglass collects telemetry from devices, types it into owned datapoints, models
health across systems and locations, alarms on it, and acts. It is a single Go binary
over a BYO PostgreSQL database, and it is two things at once on purpose: a working tool,
and a learning tool for how one is built.

:::note[Early, and public from the first commit]
Omniglass is being rebuilt in the open, one vertical slice at a time. The architecture is
published ahead of the code, so the design is readable before the feature that implements
it lands. Expect the surface to move.
:::
